# Timeline MVP - Smart Ngangon

## MVP Scope Definition

MVP fokus pada fitur inti yang memberikan nilai langsung kepada pengguna.

### Must Have (P0)
- [x] Landing page dengan informasi produk
- [x] Marketplace katalog kambing
- [x] Checkout via WhatsApp
- [x] Dashboard dengan live camera
- [x] Kontrol pakan manual
- [x] Deteksi kambing (AI basic)

### Should Have (P1)
- [x] Filter dan pencarian marketplace
- [x] Peta zonal kandang
- [x] Log aktivitas
- [x] Notifikasi sistem
- [ ] Pakan otomatis terjadwal

### Nice to Have (P2)
- [ ] Admin panel lengkap
- [ ] Multi-kandang support
- [ ] Laporan dan analytics
- [ ] Mobile app
